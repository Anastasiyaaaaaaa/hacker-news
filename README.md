# hacker-news

## О приложении 
hacker-news-app -- приложение,  которое отображает последние 100 актуальных новостей.

## Запуск приложения 
```sh
git clone https://github.com/Anastasiyaaaaaaa/hacker-news.git
cd hacker-news
yarn-start
```

## Используемые технологии
- React + Redux
- API Hacker News
- React Router v5
- Ant Design
- yarn
